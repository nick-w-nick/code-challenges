# Check Duplicates

Check for duplicates within a string and return true if a duplicate character is found, false if not.

## Example

**Input:** `a`

**Output:** `false`

</br>

**Input:** `aa`

**Output:** `true`

</br>

**Input:** `abc`

**Output:** `false`

</br>

**Input:** `abca`

**Output:** `true`
